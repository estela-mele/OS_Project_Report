Message Header Removal Script Analysis

Script Purpose

The behead.sh script removes email/news message headers, leaving only the message body.

Key Features

  • Removes everything before first blank line (standard header delimiter)
  • Handles both piped input and file arguments
  • Additional empty line removal
  • Simple and portable shell implementation

Example Transformation

Input Message:

From: sender@example.com
To: recipient@example.com
Date: Thu, 01 Jan 1970 00:00:00 +0000
Subject: Sample

This is the actual message content.

Output:

This is the actual message content.

Enhanced Version Features

  • Option to preserve Subject line
  • Verbose mode for error reporting
  • Better error handling
  • Support for more header formats
  • Command-line option parsing

Common Message Headers

Typical headers processed by this script:

  • From:, To:, Cc:, Bcc:
  • Subject:
  • Date:
  • Message-ID:
  • Content-Type:

Modern Email Processing

For more sophisticated email processing consider:

# Using formail (procmail package)
formail -X "" < message.txt  # Remove all headers

# Using Python's email module
python3 -c "import email; \
msg = email.message_from_file(open('message.txt')); \
print(msg.get_payload())"
